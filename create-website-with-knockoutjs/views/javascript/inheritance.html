Object Oriented Programming in JavaScript

In this article we are going to learn how we can do Object oriented programming in JavaScript. We can Implement Inheritance, Encapsulation, Abstraction and Polymorphism concept.

So when we talk about Object oriented programming then there are concepts of "Class, Object, Property of class, Method, Constructor etc".

Assuming you guys are aware of all aforesaid concept. If not then please go through them and then it would be very easy for you to understand.

To understand Class, Method, Object, Constructor and Property Please refer my article "Object Creation in JavaScript".

Class: In JavaScript there is no keyword 'class' unlike you found in other languages such as C++ or Java. In JavaScript defining a class is same as defining function.

eg. MyClass = function(){};

Note**: ECMA 6 onward 'class, extends etc.' keyword got added in JavaScript.

Object: Instance of Class is an Object. so when we say "var myObj = new MyClass();" then myObj is Object of MyClass.

Method: Any function part of class is method. see the example below.

Constructor: In JavaScript there is no seprate function to create constructor like we do in C++ or Java. In JavaScript the function serves as theitself act as the Constructor of object. So in JavaScript there is no need to define constructor explicitly.

Property: any attributes you create as part of the function or so called Class are Property of that class. Will see it in example.
